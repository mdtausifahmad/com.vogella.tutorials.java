=== Eclipse IDE configuration

To simplify the usage of the AssertJ `assertThat` statement in the Eclipse IDE go to menu:Window[Preferences>Java>Editor>Content assist>Favorites>New Type] and enter `org.assertj.core.api.Assertions`.


You should see org.assertj.core.api.Assertions.* in the list.

=== IntelliJ Idea configuration 

No special configuration, just start typing `assertThat` and then invoke completion (Ctrl-Space) twice.
